# 開発環境構築手順
1. プロジェクトのルートディレクトリで以下を実施する
    ```
    $ npm install
    ```
2. strongloopをインストール
    ```
    $ npm install -g strongloop
    ```
3. MongoDBをHome Brewからインストールする
    ```
    $ brew install mongodb
    ```
4. MongoDBをサービスへ登録し、PC起動時に起動するようにする
    ```
    $ brew services start mongodb
    ```

# サーバのスタート
2. プロジェクトのルートディレクトリで以下を実行する
    ```
    $ npm start
    ```
